# Fastag Fraud Detection System

This project is a Fastag Fraud Detection System built using Streamlit, pandas, and scikit-learn. The system analyzes various transaction parameters to detect potential fraud.

## Prerequisites

Before you begin, ensure you have met the following requirements:

- You have installed [Python 3.x](https://www.python.org/downloads/).
- You have installed [Git](https://git-scm.com/downloads).
- You have installed [Visual Studio Code](https://code.visualstudio.com/).

## Cloning the Repository

To clone the repository, follow these steps:

1. Open Visual Studio Code.
2. Open the Command Palette (Ctrl+Shift+P) and select `Git: Clone`.
3. Enter the repository URL:

    ```sh
    https://github.com/gaurav0973/minor_project_semester_6.git
    ```

4. Select a folder to clone the repository into.
5. Open the cloned repository in Visual Studio Code.

## Setting Up the Environment

To set up the environment, follow these steps:

1. Open a terminal in Visual Studio Code (Ctrl+`).
2. Create a virtual environment:

    ```sh
    python -m venv .venv
    ```

3. Activate the virtual environment:

    - On Windows:

        ```sh
        .venv\Scripts\activate
        ```

    - On macOS/Linux:

        ```sh
        source .venv/bin/activate
        ```

4. Install the required dependencies:

    ```sh
    pip install -r requirements.txt
    ```

## Running the Application

To run the application, follow these steps:

1. Ensure the virtual environment is activated.
2. Run the Streamlit app:

    ```sh
    streamlit run app.py
    ```

3. Open your web browser and navigate to the URL provided by Streamlit (usually `http://localhost:8501`).
